## Изменения

### Добавлено
- **VERSION файл** (текущая версия 0.0.3) - единый источник правды для версии
- **Команда `version`** - показывает текущую версию агента
- **Отображение версии** в:
  - Help (--help)
  - Status (status)
  - Version (version)
- **Скрипт auto_version_update.py** - автоматически инкрементирует версию после merge PR

### Как это работает

1. При merge этого PR версия будет 0.0.3
2. После merge можно запустить `python scripts/auto_version_update.py` для обновления до 0.0.4
3. Версия отображается во всех основных командах CLI

### Тестирование

```bash
$ python -m agent version
Shket Research Agent v0.0.3

$ python -m agent --help
usage: python -m agent [-h] {run,bot,status,version,logs} ...

Shket Research Agent v0.0.3 — автономный LLM‑агент для Ubuntu‑сервера

$ python -m agent status
Agent status: idle (v0.0.3)
Available tools: shell, filesystem, web_search, todo, backup, run_tests, run_agent_subprocess, git, request_restart
```

### Скрипт обновления версии

```bash
# Инкрементировать patch версию (0.0.3 -> 0.0.4)
python scripts/auto_version_update.py --patch

# Инкрементировать minor версию (0.0.3 -> 0.1.0)
python scripts/auto_version_update.py --minor

# Инкрементировать major версию (0.0.3 -> 1.0.0)
python scripts/auto_version_update.py --major
```
